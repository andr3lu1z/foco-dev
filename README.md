# 🎯 Foco-Dev (Rastreador de Metas)

Projeto acadêmico desenvolvido com arquitetura em 3 camadas (Front-end, Back-end e Banco de Dados).

## 🛠️ Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, JavaScript Vanilla
* **Back-end:** Node.js, Express, CORS
* **Banco de Dados:** PostgreSQL

## 🚀 Como rodar o projeto localmente

1. Crie um banco de dados chamado `foco_dev` no pgAdmin.
2. Execute o script do arquivo `banco.sql` para criar a tabela.
3. No arquivo `servidor.js`, altere a senha de conexão para a sua senha do PostgreSQL.
4. Abra o terminal na pasta do projeto e rode `npm install` para instalar as bibliotecas.
5. Rode `node servidor.js` para iniciar a API.
6. Abra o arquivo `index.html` (dentro da pasta frontend) no navegador.
